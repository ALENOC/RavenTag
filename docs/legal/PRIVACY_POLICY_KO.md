## 3. 수집하는 데이터와 사용 방법

### 3.1 기기에만 저장되는 데이터 (전송 없음)

다음 데이터는 기기에만 저장되며 개발자가 운영하는 서버로 전송되지 않습니다:

| 데이터 | 목적 | 저장 위치 |
|---|---|---|
| BIP39 니모닉 구문 (암호화) | 지갑 복구 | Android Keystore (AES-256-GCM) |
| 개인 키 (파생, 암호화) | 거래 서명 | Android Keystore (AES-256-GCM) |
| 지갑 주소 (RVN) | 표시 및 거래 | 로컬 암호화 저장소 |
| 관리자/운영자 키 (Brand 버전) | 자산 관리 | Android Keystore (AES-256-GCM) |
| 앱 설정 및 환경설정 | 앱 구성 | 로컬 공유 환경설정 |

**니모닉 구문과 개인 키는 절대 기기를 벗어나지 않습니다.**

### 3.2 NFC 태그 인증 중 전송되는 데이터

NFC 태그를 스캔하면 앱은 암호화 검증을 수행하기 위해 다음 데이터를 백엔드 서버로 전송합니다:

| 데이터 | 목적 |
|---|---|
| 자산 이름 (예: BRAND/PRODUCT#001) | 블록체인에서 자산 식별 |
| 암호화된 NFC 카운터 (e 매개변수) | SUN MAC 검증 |
| NFC MAC 값 (m 매개변수) | SUN MAC 검증 |
| 기기 IP 주소 | 서버 측 요청 제한 및 보안 로깅 |

이 데이터는 NFC 태그의 진품 여부를 확인하는 데 필요한 최소한의 데이터입니다. 검증 요청에는 IP 주소 외에 개인 식별 정보가 포함되지 않습니다.

**어느 서버가 이 데이터를 수신하는지는 앱 구성에 따라 다릅니다:**

- **기본/RavenTag 운영 백엔드**: 앱이 개발자가 운영하는 RavenTag 백엔드 (raventag.com)에 연결되는 경우, 개발자는 본 개인정보 처리방침에 설명된 대로 이 데이터를 수신하고 처리합니다.
- **브랜드 운영 백엔드**: 앱이 자체 서버에 연결되도록 브랜드가 컴파일하거나 구성한 경우, 해당 브랜드의 서버가 이 데이터를 수신합니다. 개발자는 이 데이터를 수신, 접근 또는 처리하지 않습니다. 브랜드가 독립적인 개인정보 처리자가 되며 자체 개인정보 처리방침이 적용됩니다.

앱 설정에 표시된 서버 URL을 확인하여 앱이 어느 백엔드에 연결되는지 확인할 수 있습니다.

**보존 기간 (개발자 운영 백엔드)**: IP 주소와 요청 로그는 보안 및 요청 제한 목적으로 최대 90일간 보존 후 자동 삭제됩니다.

**법적 근거 (GDPR, 개발자 운영 백엔드)**: 정당한 이익 (GDPR 제6조(1)(f)): 보안 모니터링 및 사기 방지.

### 3.3 블록체인 작업 중 전송되는 데이터

지갑 작업(잔액 확인, RVN 전송, 자산 발행)을 수행하면 앱이 Ravencoin 네트워크 노드와 통신합니다. 이 통신에는 다음이 포함될 수 있습니다:

| 데이터 | 목적 |
|---|---|
| Ravencoin 지갑 주소 | 잔액 및 거래 내역 조회 |
| 거래 데이터 | 네트워크에 거래 브로드캐스트 |
| 기기 IP 주소 | 네트워크 통신 |

Ravencoin 블록체인은 공개 분산 네트워크입니다. 네트워크에 브로드캐스트된 모든 거래는 누구나 영구적으로 공개 열람할 수 있습니다. 비공개로 유지하고 싶은 거래에는 이 지갑을 사용하지 마세요.

### 3.4 자산 이미지 로드 중 전송되는 데이터

IPFS에 호스팅된 자산 이미지를 로드할 때 앱은 공개 IPFS 게이트웨이 (ipfs.io, cloudflare-ipfs.com 등)에 연결합니다. 이러한 제3자 서비스는 자체 개인정보 처리방침에 따라 IP 주소를 기록할 수 있습니다.

### 3.5 카메라 데이터

앱 내에서 QR 코드를 스캔하기 위해 카메라를 사용하는 경우, 카메라 데이터는 기기에서만 실시간으로 처리되며 저장되거나 전송되지 않습니다.

### 3.6 NFC 데이터

NFC 태그 데이터는 기기에서 로컬로 읽힙니다. 원시 NFC 데이터 (UID, NDEF 레코드)는 기기에서 처리되며 파생된 검증 매개변수 (자산, e, m)만 제3.2항에 설명된 대로 전송됩니다.

### 3.7 수집하지 않는 데이터

다음은 명시적으로 수집하지 않습니다:

- 이름, 이메일 주소 또는 기타 개인 식별 정보.
- 기기 식별자 (IMEI, Android ID, 광고 ID).
- 위치 데이터.
- 사용 분석 또는 텔레메트리.
- 충돌 보고서 (귀하가 명시적으로 제출한 경우 제외).
- 광고 목적의 데이터.

## 4. 제3자 서비스

앱은 다음 제3자 서비스와 상호작용합니다. 이들의 개인정보 처리방침이 데이터 처리 방식을 규율합니다:

| 서비스 | 목적 | 개인정보 처리방침 |
|---|---|---|
| Ravencoin 네트워크 노드 | 블록체인 조회 및 거래 | 분산형 네트워크, 단일 정책 없음 |
| IPFS 게이트웨이 (ipfs.io, cloudflare-ipfs.com) | 자산 이미지 로드 | 각 제공업체 참조 |
| Pinata (pinata.cloud) | IPFS 메타데이터 고정 (선택사항, Brand 버전만 해당) | https://pinata.cloud/privacy |
| Google Play Store | 앱 배포 | https://policies.google.com/privacy |

개발자는 이러한 제3자 서비스의 데이터 처리 방식에 대해 책임을 지지 않습니다.

## 5. 데이터 보안

기기에 저장된 모든 민감한 데이터 (니모닉 구문, 개인 키, API 키)는 사용 가능한 경우 하드웨어 지원 보안을 사용하는 Android Keystore 시스템을 통해 AES-256-GCM으로 암호화됩니다.

앱과 개발자 백엔드 서버 간의 통신은 HTTPS/TLS로 암호화됩니다.

이러한 조치에도 불구하고 전자 저장 또는 전송 방법은 100% 안전하지 않습니다. 기기와 니모닉 구문의 보안 유지는 귀하의 책임입니다.

## 6. 데이터 보존

- **기기 상의 데이터**: 지갑을 삭제하거나 앱을 제거할 때까지 저장됩니다.
- **서버 측 요청 로그**: 최대 90일 보존 후 자동 삭제됩니다.
- **블록체인 데이터**: Ravencoin 블록체인에 브로드캐스트된 모든 거래는 영구적으로 공개되며 개발자나 제3자가 삭제할 수 없습니다.

## 7. GDPR에 따른 귀하의 권리

유럽 경제 지역에 거주하는 경우 개인 데이터에 관해 다음과 같은 권리가 있습니다:

- **접근권**: 보유 중인 개인 데이터 사본 요청 (서버 측 로그에 한함).
- **정정권**: 부정확한 데이터 정정 요청.
- **삭제권**: 법적 보존 의무에 따라 서버 (서버 로그)에서 개인 데이터 삭제 요청.
- **처리 제한권**: 데이터 사용 방식 제한 요청.
- **이의 신청권**: 정당한 이익에 기반한 처리에 이의 신청.
- **데이터 이동성권**: 구조화된 기계 가독 형식으로 데이터 수령.
- **불만 신청권**: 이탈리아 개인정보 보호 기관 (Garante per la protezione dei dati personali, https://www.garanteprivacy.it)에 불만을 신청할 권리가 있습니다.

이러한 권리를 행사하려면 다음으로 문의하세요: https://github.com/ALENOC/RavenTag / legal@raventag.com

앱은 만 18세 미만의 아동을 대상으로 하지 않습니다. 만 18세 미만의 아동으로부터 의도적으로 개인 데이터를 수집하지 않습니다. 만 18세 미만의 아동이 앱을 사용하여 개인 데이터를 제공했다고 생각되는 경우 문의하시면 해당 데이터를 삭제하는 조치를 취하겠습니다.

## 9. 국제 데이터 이전

**개발자가 운영하는 데모 백엔드**: 개발자의 데모 백엔드 서버는 미국 또는 유럽연합 내에 위치합니다. EU 외부에서 데모 앱 인스턴스에 접근하는 경우, 검증 요청 데이터 (제3.2항)는 GDPR 요건에 따라 EU로 이전되어 미국 또는 EU 내에서 처리됩니다.

**브랜드가 운영하는 백엔드**: 프로덕션 배포에서 백엔드 서버의 지리적 위치는 전적으로 배포한 브랜드 또는 제조업체에 의해 결정됩니다. 개발자는 제3자 브랜드가 선택한 서버 위치를 통제하거나 알지 못합니다. 적용 가능한 국제 데이터 이전 규칙은 해당 배포를 운영하는 브랜드의 규칙입니다. 자세한 내용은 브랜드의 개인정보 처리방침을 참고하세요.
